# IFC 3Dビューア — 要件定義書

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | CAD Viewer（IFC 3Dビューア） |
| 配置 | `ccagi/cad/` |
| 目的 | IFC（Industry Foundation Classes）ファイルをアップロードし、ブラウザ上で3Dモデルを閲覧・操作できるBIMビューアWebアプリ |
| 対象ユーザー | 建築設計者、施工管理者、BIMコーディネーター |

## 2. 機能要件

### 2.1 認証機能
- メールアドレス + パスワードによるログイン
- 新規アカウント登録
- ログアウト
- 認証状態によるルート保護（未ログイン時はログイン画面へリダイレクト）

### 2.2 プロジェクト管理
- プロジェクトの作成（名前・説明）
- プロジェクト一覧表示（ステータス: active / archived）
- プロジェクト詳細（所属ファイル一覧）
- プロジェクトの削除（関連ファイルもカスケード削除）

### 2.3 ファイル管理
- IFCファイルのアップロード（ドラッグ&ドロップ対応）
- ファイル一覧表示（ファイル名・サイズ・バージョン・アップロード日）
- ファイルバージョン管理
- ファイルメモ

### 2.4 3Dビューア（コア機能）
- IFCファイルの3D表示（IFC2X3 / IFC4対応）
- カメラ操作（回転・ズーム・パン）
- ビュー切替（全体表示・正面・上面・側面）
- ワイヤーフレーム表示切替

### 2.5 モデルツリー
- IFC構造の階層ツリー表示
- 要素クリックでカメラフォーカス

### 2.6 プロパティパネル
- 選択要素の基本情報（名前・タイプ・Express ID）
- ジオメトリ情報（位置）
- バウンディングボックスサイズ

### 2.7 ダッシュボード
- プロジェクト数・ファイル数の統計表示
- 最近アップロードされたファイルの一覧
- 各ファイルへのクイックアクセス

### 2.8 ユーザー設定
- 表示名の変更

## 3. 非機能要件

| 項目 | 要件 |
|------|------|
| レスポンス | ページ遷移 2秒以内 |
| IFC読込 | 10MB以下のファイルで30秒以内 |
| ブラウザ対応 | Chrome, Edge, Firefox（最新版） |
| セキュリティ | RLS（Row Level Security）による全テーブルのアクセス制御 |
| 認証 | Supabase Auth（JWT） |
| ストレージ | Supabase Storage（認証済みユーザーのみアクセス可） |

## 4. テスト用機能
- `/test-sample` — ローカルサンプルIFCファイルの自動読み込み（認証不要）
- `/test-viewer` — ドラッグ&ドロップでIFCファイルを読み込み（認証不要）
