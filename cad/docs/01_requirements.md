# 3Dモデルビューア — 要件定義書

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | CAD Viewer（3Dモデルビューア） |
| 配置 | `ccagi/cad/` |
| 目的 | 複数形式の3Dモデルファイルをアップロードし、ブラウザ上で3Dモデルを閲覧・操作できるビューアWebアプリ |
| 対象ユーザー | 建築設計者、施工管理者、BIMコーディネーター、3Dモデル制作者 |

## 2. 対応フォーマット

| 拡張子 | 形式名 | ローダー |
|--------|--------|---------|
| `.ifc` | IFC (Industry Foundation Classes) | web-ifc (WASM) |
| `.glb` / `.gltf` | glTF / glTF Binary | GLTFLoader |
| `.fbx` | Autodesk FBX | FBXLoader |
| `.obj` | Wavefront OBJ | OBJLoader |
| `.stl` | STL (3Dプリント) | STLLoader |
| `.dae` | COLLADA | ColladaLoader |
| `.ply` | PLY (Point Cloud) | PLYLoader |
| `.3ds` | 3D Studio | TDSLoader |

各ローダーは dynamic import により必要時のみロード。

## 3. 機能要件

### 3.1 認証機能
- メールアドレス + パスワードによるログイン
- 新規アカウント登録
- ログアウト
- 認証状態によるルート保護（未ログイン時はログイン画面へリダイレクト）

### 3.2 プロジェクト管理
- プロジェクトの作成（名前・説明）
- プロジェクト一覧表示（ステータス: active / archived）
- プロジェクト詳細（所属ファイル一覧）
- プロジェクトの削除（関連ファイルもカスケード削除）

### 3.3 ファイル管理
- 3Dファイルのアップロード（ドラッグ&ドロップ対応、全対応形式）
- ファイル一覧表示（ファイル名・サイズ・バージョン・アップロード日）
- ファイルバージョン管理
- ファイルメモ

### 3.4 3Dビューア（コア機能）
- マルチフォーマット3D表示（9形式対応）
- カメラ操作（回転・ズーム・パン）
- ビュー切替（全体表示・正面・上面・側面）
- ワイヤーフレーム表示切替

### 3.5 モデルツリー
- モデル構造の階層ツリー表示（IFC: 要素タイプ / 他形式: Object3D階層）
- ノードクリックでカメラフォーカス + 青色ハイライト + プロパティ表示

### 3.6 プロパティパネル
- 選択要素の基本情報（名前・タイプ・ID）
- ジオメトリ情報（位置）
- バウンディングボックスサイズ
- Groupノードの場合は子要素数も表示

### 3.7 パネル連動
- 3Dモデルクリック → ツリーハイライト + プロパティ更新
- ツリーノードクリック → カメラ移動 + 3Dハイライト + プロパティ更新
- 空クリック → 全選択解除

### 3.8 ダッシュボード
- プロジェクト数・ファイル数の統計表示
- 最近アップロードされたファイルの一覧
- 各ファイルへのクイックアクセス

### 3.9 ユーザー設定
- 表示名の変更

## 4. 非機能要件

| 項目 | 要件 |
|------|------|
| レスポンス | ページ遷移 2秒以内 |
| モデル読込 | 10MB以下のファイルで30秒以内 |
| ブラウザ対応 | Chrome, Edge, Firefox（最新版） |
| セキュリティ | RLS（Row Level Security）による全テーブルのアクセス制御 |
| 認証 | Supabase Auth（JWT） |
| ストレージ | Supabase Storage（認証済みユーザーのみアクセス可） |

## 5. テスト用機能
- `/test-sample` — ローカルサンプルファイルの自動読み込み（認証不要）
- `/test-viewer` — ドラッグ&ドロップ or ファイル選択で3Dファイルを読み込み（認証不要、全形式対応）
